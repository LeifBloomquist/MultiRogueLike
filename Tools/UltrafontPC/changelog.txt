CHANGELOG

UltrafontPC
A cross development tool for creating
and editing C64 Hires charsets

Based on VICZen
A cross development char/tile/map editor
for the C64

VICZen 0.6a by Mark Thomas Ross (AuMTRtoN)

VICZen to UltrafontPC fixes, changes and
additional code by Andrew Vaisey (aNdy)

Arkanix Labs (http://www.arkanixlabs.com)
